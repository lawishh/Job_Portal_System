# zidio-connect

Simple starter scaffold for zidio-connect:

- backend: Spring Boot (in-memory repositories) in `backend/`
- frontend: static HTML/JS in `frontend/`

How to run backend (requires Java 21 and Maven):

```powershell
cd zidio-connect\backend
mvn spring-boot:run
```

Then open `zidio-connect/frontend/index.html` in a browser (or serve it with any static server). The frontend calls the backend at `http://localhost:8080/api`.
